  9. We would like to point out the following limitations:
    1. We did not implement seemless recovery from a power outage.
    2. We did not implement a buffer to handle cases where there is a large amount of containers on a ship.
    3. We did not implement duplicate name checking of containers. The container the operator selects will be the one to be unloaded. Furthermore, animations will struggle when loading containers with duplicate names.
